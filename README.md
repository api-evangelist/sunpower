# SunPower

SunPower is a residential solar energy company (Nasdaq: SPWR) that designs, installs, and finances solar systems and battery storage for homeowners. The original SunPower Corporation filed for Chapter 11 bankruptcy in August 2024; its assets were acquired by Complete Solaria in September 2024, which rebranded as SunPower Inc. in April 2025.

SunPower maintains the GitHub organization [github.com/SunPower](https://github.com/SunPower) hosting open-source solar modeling tools used by PV professionals and researchers.

## Open Source Libraries

- **[pvfactors](https://github.com/SunPower/pvfactors)** — View-factor model for diffuse shading and bifacial PV modeling ([docs](https://sunpower.github.io/pvfactors/), [PyPI](https://pypi.org/project/pvfactors/))
- **[PVMismatch](https://github.com/SunPower/PVMismatch)** — Mismatch loss calculations for PV cells, modules, and strings
- **[SolarUtils](https://github.com/SunPower/SolarUtils)** — Solar position algorithms and irradiance utility functions
